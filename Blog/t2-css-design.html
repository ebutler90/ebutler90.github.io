Covering: Static, Relative, Absolute, and Fixed Positioning

Monday, January 4, 2015

Hello Again! The topic we'll be covering today is: The difference between relative, absolute, and fixed positioning.

Trying to understand the positioning of elemtents was one of the harder concepts for me to grasp, so I did some extra research to try to teach you the basics. As they say, teaching is the best way to learn!

So the most important thing to understand when creating a webpage is that every element if just a block of pixels. What you're aiming to do with the positioning is move that block of pixels exactly where you want it on a page. There are a few differenet ways to do that.

Static Positioning
Setting an element to position:static essentially means you want the element to stay exactly where it would normally fall on the page. It remains in its default state. This is a position that is rarely used, but it is definitely important to know that it exists.
Relative Positioning
When an element is set to position:relative, you are setting the element to be moved around relative to it's normal (or static) position. If you just leave it as position:relative, and dont specify values to the top, bottom, left, or right properties, it won't move at all. You use those properties to move the element around relative to where it would be. Positioning an element as relative is used often as a block for absolutely positioned elements because it limits the position of child elements that are set to absolute.
Absolute Positioning
When you set an element to position:absolute, you have the ability to place it exactly where you need it. Absolute elements will be relative to any parent element that is not static. If there is no such element, it will default to being relative to , which essentially means it's relative to the webpage itself. (This is why I mentioning absolute positioning when talking about relative positioning. Relative positioned elements will block in absolutely positionined elements.)
Fixed Positioning
The last form of positioning available is position:fixed. This positioning will position an element relative to the actual browser window. This means that the item wont move when the user scrolls through the content.
